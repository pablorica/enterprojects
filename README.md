## Download and Install

Let's go!

```
$ git clone https://github.com/pablorica/enterprojects.git
$ cd enterprojects
$ yarn // or npm install
```

## Development

Run development page on **localhost:8080**

```
$ npm run dev
```

## Build

Build for production.

```
$ npm run build
```
